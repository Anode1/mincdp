The moment

It could build the thing, but it couldn’t see it.

Mid-task on a private project, a coding agent finished a waveform-rendering feature and then stopped short. It could reason about the DOM, but it had no way to confirm that pixels actually landed on the screen the way a person checks. So it hedged — and I told it I’d already built it a way to look.

Anatomy

Two kinds of eyes.

The distinction is the whole idea: seeing a result, versus using an interface and then seeing what you did.

01 · Seeing

The frozen glance

Render the page, capture a picture, let the agent read the pixels. The direct heir of screen-scraping — a human glance replaced by a PNG the agent can actually parse. tests/shot.sh.

02 · Acting

The live page

Not a snapshot but a session: type, press Enter, assert, then screenshot the driven state. The DevTools Protocol doing what Xvfb screen-scraping used to fake. cdp_screenshot.

Honesty about prior art

I did not invent agent vision. The repo ships the proof.

Alongside the modern code, mincdp keeps my 2005 Xvfb ancestor as a period piece: start a real Firefox against a virtual framebuffer, grab the whole root window with xwd, thumbnail it with ImageMagick, orchestrate the batch in Java. Two decades apart, both do the same thing — render, then capture a picture.

The idea of letting a machine see a rendered page is old. What is new is not the screenshot; it is that the thing reading the screenshot is an agent in the middle of writing the code, using the picture to decide whether to commit.

The claim, stated precisely

Not: I invented agent vision. I didn’t — and my own 2005 folder proves the lineage runs deep.

Just: I independently built a clean, minimal implementation, wired it into a real agent’s verify-before-commit loop, and open-sourced it on Jul 1 — with the origin dated Jun 25 in a WhatsApp thread and the code dated in git.

What I’m asking for

Nothing but attribution. Not money, not a dispute, not a takedown — just the record of where and when this was figured out, and by whom. If the idea travels, and I hope it does, a mention is all I want.
